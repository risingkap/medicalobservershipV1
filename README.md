# Setup Instructions

### 1. Prerequisites

- Ensure you have the **.NET 8.0 SDK** installed on your machine.

### 2. Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/risingkap/medicalobservershipV1.git
    cd medicalobservershipV1
    ```
2.  Restore dependencies:
    ```bash
    dotnet restore
    ```

### 3. Running the Application

1.  Start the development server:
    ```bash
    dotnet run --project medicalobservershipV1/medicalobservershipV1.csproj
    ```
2.  Open your browser and navigate to:
    `http://localhost:5121`
